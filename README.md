- Node 18.15.0

# Project Setup

### Access project folder and and download dependencies

```sh
cd BidBuddy
cd Frontend
npm install
```

### Start Project in Development Environment

```sh
npm run dev
```

### Build Vite Project

```sh
npm run build
```

### Preview Vite Project

```sh
npm run preview
```

# Backend

### Build and Start All Docker Containers

```sh
docker-compose up -d
```

### Stop All Docker Containers

```sh
docker-compose down
```
